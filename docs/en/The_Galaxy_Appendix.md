**\[Corollary\] Re = 0 → no time.** ← If nothing to resolve, no order is
needed. No order = no time. ∎

**\[Corollary\] Re \> 0 → time exists.** ← Components are entangled and

← Euler’s formula e^(iθ) traces a circle. Return to +1 after 2π is an

**\[Theorem\] Time is neither a line nor a circle, but a helix

**\[Correspondence\] The four DNA bases map to the four Re components.**

| DNA base     | Base pair | Matrix type  | Re component |
|:-------------|:----------|:-------------|:-------------|

**\[Correspondence\] Chargaff’s Second Parity Rule (1968) = algebraic
symmetry of Re.**

58-year-old empirical observation has no accepted causal explanation.

v8.0 interpretation: Re is a single entangled operator. The four
components (c₀, c₁, c₂, c₃) are not independent but constrained by
algebraic structure. In sufficiently long sequences (large samples),
this constraint manifests as statistical parity between paired
components.

**\[Correspondence\] CpG suppression = \[σ₁, σ₂\] = 2iσ₃ ≠ 0.**

The dinucleotide CG (CpG) is systematically underrepresented relative to

v8.0: C and G are off-diagonal (σ₁, σ₂). Since commutator \[σ₁, σ₂\] =
2iσ₃ ≠ 0, order matters. CG ≠ GC (algebraically). Biological

# Part E: Experimental Limitations (Honest Record)

1.  **No independent data collection.** All results are based on
    published data, public databases, or computational generation.
    Appropriate as a first-step correspondence check but does not
    constitute independent empirical verification.

2.  **Self-evaluation bias** (Session 1 AI experiments): gpt-4o
    generated and evaluated its own output.

3.  **Literature-based physiological data** (cell cycle, HRV, sleep,
    emotion, brain): Based on published studies, not independent
    measurements. Confirms direction and structure but is not
    independent.

4.  **Small sample sizes**: DNA Phase 1 (N=12 species), Language Session
    1 (N=10), cell cycle (N=10 cell types), protein (N=18 structures).
    Statistical power is limited in several domains.

5.  **Post-hoc reinterpretation**: A2 (vowel asymmetry), C3 (market
    volume), H3 (amino acid mapping revision), N3 (protein conservation
    direction) were reinterpreted after initial prediction failure.
    Clearly marked throughout and should not carry the same weight as
    confirmed predictions.

6.  **Confirmation bias risk**: The investigator (v8.0 framework
    developer) designed predictions and evaluated results. Independent
    replication by researchers unfamiliar with v8.0 would significantly
    strengthen these findings.

7.  **Structural correspondence ≠ causal mechanism**: Observing the same
    algebraic patterns across multiple domains proves structural
    isomorphism, not causal connection. Each domain has its own causal
    mechanisms (methylation in DNA, dopamine in brain networks,
    gravitational attraction in economics). v8.0 identifies a shared
    mathematical skeleton; it does not replace domain-specific
    explanations.

8.  **Failed predictions included without concealment**: P5 (conserved
    sequences), A2 (vowel symmetry), C3 (market volume), N3 (protein
    conservation), M2 (convergence rate). Honest reporting of failures
    is essential to scientific integrity.
